# Canvas Game Starter (Vite + TypeScript)

Modern web game development setup with hot reload.

## Setup

```bash
npm install
npm run dev
```

Open browser to http://localhost:5173

## Project Structure

```
src/
  main.ts       - Game loop
  game.ts       - Game class
  style.css     - Styles
index.html      - Entry point
```

## Features

- TypeScript for type safety
- Hot module reload
- Canvas 2D rendering
- Game loop with delta time
- Input handler example

## Build for Production

```bash
npm run build
# Output in dist/
```
